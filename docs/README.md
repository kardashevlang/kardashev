# kardashev documentation

Guides in this directory:

- **[Getting started](getting-started.md)** — install, first program,
  projects & `build.ks`, tests, formatting, docs, cross-compiling.
- **[Language tour](language-tour.md)** — every feature of v0.179.0 with
  runnable examples.
- **[Standard library](stdlib.md)** — the embedded `@import("std")` modules:
  containers, algorithms, text, JSON/base64/hex, hashes, glob, RNG.
- **[Self-hosting](selfhosting.md)** — the compiler-in-kardashev under
  `selfhost/` and its differential-testing discipline.

Elsewhere in the repository:

- [`SPEC.md`](../SPEC.md) — the **normative** language & toolchain
  specification (version-annotated).
- [`examples/`](../examples/) — 41 single-feature example programs
  ([index](../examples/README.md)).
- [`ROADMAP-RUST-ZIG.md`](../ROADMAP-RUST-ZIG.md) — the Generation-2 plan
  and shipped history; [`CHANGELOG.md`](../CHANGELOG.md) — per-version
  notes.
- [`CONTRIBUTING.md`](../CONTRIBUTING.md) — workflow, test suites,
  conventions; [`SECURITY.md`](../SECURITY.md) — reporting vulnerabilities.

The rendered documentation website lives at
**<https://kardashevlang.org>** (source:
[kardashevlang/kardashevlang.org](https://github.com/kardashevlang/kardashevlang.org)).
